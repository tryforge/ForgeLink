# $previousTrack
Goes to a previous track in the queue (if one exists)
## Usage
```
$previousTrack[guildID]
```
## Fields
|   Name   |     Description     | Type  | Required | Rest |
|----------|---------------------|-------|----------|------|
| Guild ID | The ID of the guild | Guild | Yes      | No   |

## Output
> String
