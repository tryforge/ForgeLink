# $getNodes
Gets Player Nodes
## Usage
```
$getNodes[guildID]
```
## Fields
|   Name   |              Description              | Type  | Required | Rest |
|----------|---------------------------------------|-------|----------|------|
| Guild ID | The ID of the guild of the player to. | Guild | Yes      | No   |

## Output
> Boolean
