# $nodeExists
Gets Player Nodes
## Usage
```
$nodeExists[name]
```
## Fields
| Name |     Description      |  Type  | Required | Rest |
|------|----------------------|--------|----------|------|
| Name | The name of the node | String | Yes      | No   |

## Output
> Boolean
